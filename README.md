# Claude Skills

Кастомные команды (скилы) для Claude Code.

## Установка

Скопируй нужные файлы из папки `commands/` в `~/.claude/commands/`:

```bash
cp commands/creo.md ~/.claude/commands/creo.md
```

После этого команда доступна в Claude Code как `/creo`.

---

## /creo — Генератор рекламных сценариев

Пишет 10 готовых к съёмке рекламных скриптов для Meta (Reels/Stories) по любой теме.

**Использование:**
```
/creo онлайн-курс по фотографии, 49€, ЦА — новички 25-40, язык — русский
/creo нутрициолог, программа похудения, ЦА — женщины 30-45
/creo веб-разработка, буткемп
```

**Что внутри:**
- Структура скрипта из 8 шагов
- 11 ключевых правил написания
- 10 обязательных углов (FOMO, before/after, антихайп, прокрастинация и др.)
- 5 форматов съёмки с режиссёрскими пометками
- Чеклист проверки каждого скрипта

## Обновление

Чтобы получить последнюю версию:

```bash
git pull

```
