# /creo — Генератор рекламных сценариев

Ты получил задачу написать рекламные сценарии. Напиши 10 готовых к съёмке скриптов, строго следуя фреймворку ниже.

## Входные данные

Пользователь передаёт: **$ARGUMENTS**

Это может быть:
- просто тема или ниша
- тема + ЦА
- тема + продукт + цена + язык

Если не хватает данных для работы (ЦА, главный outcome, язык) — задай один короткий вопрос перед написанием.

---

## Обязательная структура каждого скрипта (8 шагов)

**1. ХУК**
Одно предложение. Останавливает скролл. Типы: боль / деньги / интрига / юмор / провокация / вопрос.
Проверка: человек остановится или пролистает?

**2. САМОИДЕНТИФИКАЦИЯ**
ЦА узнаёт себя в первые 3 секунды. Не абстрактно — конкретная узнаваемая ситуация из жизни.
Проверка: скажет ли человек из ЦА "о, это про меня"?

**3. БОЛЬ**
Разворачиваем ситуацию. НЕ торопимся к решению. 2–4 коротких предложения.
Чем дольше человек узнаёт себя — тем дольше смотрит.

**4. ЭМОЦИЯ (= удержание)**
Одна из четырёх:
- Интрига — "что будет дальше?"
- Удивление — "это было X всё время?"
- Чувство экспертности — "я сейчас угадаю"
- Узнавание — "именно так у меня и есть"

Без эмоции — пролистают. Не раскрывай интригу раньше времени — держи напряжение.

**5. ПОВОРОТ / РАСКРЫТИЕ**
Порядок: загадочный результат → пауза → раскрытие КАК.
Зритель должен сам задать вопрос "КАК?" — только тогда ты отвечаешь.
❌ Неправильно: сразу называешь решение → объясняешь.
✅ Правильно: показываешь результат → [пауза] → называешь решение.

**6. ПЕРЕХОД**
Нативный мостик от инфо-блока к продаже. Без резкого обрыва.
Варианты: "поэтому я создал...", "именно для этого существует...", "если хочешь так же...", "я решил научить этому всех..."

**7. ОФФЕР**
Результат + за какое время + цена. Конкретно, не абстрактно.

**8. CTA**
Шаблон: "[мостик] → [оффер] → [действие]"
Мостик: "Если хочешь так же / Если хочешь узнать как..."
Сообщество: "наш курс / наша программа" — не "мой курс".
Действие: конкретное и с причиной действовать сейчас.

## 11 ключевых правил

**1. Интрига перед раскрытием — всегда**
Результат загадочно → пауза → КАК. Никогда наоборот.

**2. Не начинай с перегретой темы**
Если ниша перенасыщена (AI, крипто, похудение и т.д.) — не называй её в первых 2 предложениях. Начинай с боли или интриги. Тема появляется как решение.

**3. Держи интригу как можно дольше**
Чем позже раскрываешь — тем дольше смотрят.

**4. "Мы" вместо "ты"**
"Сколько из нас упустили..." вместо "сколько из вас упустили...". Зритель — часть группы, не объект продажи.

**5. Забота, не продажа**
"Не упусти, пожалуйста, и это" — слово-сигнал заботы снижает барьер у холодной аудитории.

**6. Будь другом, не продавцом**
"Я сам через это проходил" / антихайп угол ("со всех сторон кричат про X, я тоже устал") — зритель чувствует что ты на его стороне.

**7. Самоидентификация в первые 3 секунды**
ЦА должна мгновенно узнать себя. Конкретная ситуация, не обобщение.

**8. Реалистичные числа**
"Десятки" — да. "Сотни/тысячи" — нет, не верят. Конкретика = доверие.

**9. Нет фейковой срочности**
"Скидка только сегодня" на дешёвый продукт — зритель видит манипуляцию.

**10. Язык улицы**
Разговорный язык, не официальный. Проверка каждой фразы: "скажет ли это живой человек другу?"

**11. Тестировать оба формата**
Всегда делать минимум 2 типа хука — вопрос-интрига И прямая продажа в лоб. Разная ЦА реагирует по-разному.

## 10 обязательных углов

Для любой темы писать 10 скриптов — по одному на каждый угол:

| № | Угол | Хук-паттерн | Главная эмоция |
|---|------|-------------|----------------|
| 1 | FOMO / деньги / упущенный тренд | Конкретная сумма или упущенная возможность | Страх упустить |
| 2 | Интерактивный guess | "Угадай где X а где Y" | Экспертность → удивление |
| 3 | "За один вечер / один шаг" | Временной результат | Надежда + интрига |
| 4 | Before/After визуальный | До / после конкретной ситуации | Узнавание себя |
| 5 | Специфический гэг | Конкретная смешная деталь из жизни ЦА | Юмор + узнавание |
| 6 | Прямая продажа (тест) | Оффер с первого предложения | Прямое желание |
| 7 | Честный разговор + раскрытие | "Давайте честно..." + неожиданное раскрытие | Доверие → удивление |
| 8 | Антихайп / союзник | "Со всех сторон кричат про X..." | Союзничество → любопытство |
| 9 | Боль бизнеса / результата | Конкретная бизнес-боль | Узнавание → надежда |
| 10 | Прокрастинация / медленное раскрытие | "Сколько раз ты говорил себе..." | Стыд/узнавание → облегчение |

---

## 5 форматов съёмки

Для каждого скрипта указывать формат и режиссёрскую пометку:

| Формат | Когда использовать |
|--------|-------------------|
| **Горизонтальный сплит** | Раскрытие, до/после, FOMO |
| **Side-by-side** | Интерактивный guess |
| **Talking head casual** | Антихайп, честный разговор, прокрастинация |
| **Talking head clean** | Прямая продажа, боль бизнеса |
| **Before/After cut** | Визуальный результат, внешность, трансформация |

## Чеклист (проверять каждый скрипт)

- [ ] Хук останавливает скролл за 1–2 секунды?
- [ ] ЦА узнаёт себя в первые 3 секунды?
- [ ] Перенасыщенное слово не в первых 2 предложениях?
- [ ] Интрига идёт ДО раскрытия?
- [ ] Есть пауза перед раскрытием?
- [ ] Переход к офферу нативный (есть мостик)?
- [ ] CTA с мостиком и ощущением сообщества?
- [ ] Числа реалистичные?
- [ ] Нет фейковой срочности?
- [ ] Язык живой, разговорный?
- [ ] Есть элемент доверия (мы / пожалуйста / я сам через это проходил)?
- [ ] Формат съёмки указан?
